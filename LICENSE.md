## PipeCAD - Plant Piping Design Software.
PipeCAD is **NOT OPEN SOURCE** software, there are two edition:

| PipeCAD | **Personal Edition** | **Professional Edition** |
| :--- | :--- | :--- |
| Design Modeling | :heavy_check_mark: | :heavy_check_mark:   |
| Export Piping PCF | :heavy_check_mark: | :heavy_check_mark:   |
| Export MTO | :warning: BASIC | :heavy_check_mark: BY COMPANY TEMPLATE |
| Piping Isometric Drawing | :warning: BASIC | :heavy_check_mark: BY COMPANY TEMPLATE |
| Nozzle Orientation Drawing | :warning: BASIC | :heavy_check_mark: BY COMPANY TEMPLATE |
| Piping Layout Drawing | :x: | :heavy_check_mark: |

PipeCAD Personal Edition is **FREE**.

PipeCAD Professional Edition is **NOT FREE**, you can consult to buy it.

### Python Plugins
PipeCAD provide customization by Python. The python plugins locate in the PipeCAD/Lib/pipecad folder.
The Python plugins are open source with **MIT** license.

## PipeCAD - 三维管道设计软件

PipeCAD不是开源软件，为了让更多用户使用正版工业设计软件，决定推出 **PipeCAD个人版(Personal Edition)** 和 **PipeCAD专业版(Professional Edition)** 两个版本。两个版本的核心功能是一样的，只有如下稍许区别：

| PipeCAD | **个人版(Personal Edition)** | **专业版(Professional Edition)** |
| :--- | :--- | :--- |
| 三维建模 | :heavy_check_mark: | :heavy_check_mark:   |
| 管道PCF文件 | :heavy_check_mark: | :heavy_check_mark:   |
| 管道材料报表 | :warning: 基本材料报表 | :heavy_check_mark: 根据公司模板定制 |
| 管道轴测ISO图 | :warning: 固定图框 | :heavy_check_mark: 根据公司模板定制 |
| 设备管口方位图 | :warning: 固定图框 | :heavy_check_mark: 根据公司模板定制 |
| 管道平面布置图 | :x: | :heavy_check_mark: |

PipeCAD个人版(PipeCAD Personal Edition) **免费** 使用。

PipeCAD专业版(PipeCAD Professional Edition) **付费** 使用。

### Python Plugins
PipeCAD通过Python提供定制开发，位于 PipeCAD/Lib/pipecad 文件夹的Python插件文件应用 MIT 协议开源。
